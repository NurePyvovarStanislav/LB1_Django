# Django Articles Project

## Опис проєкту

Цей проєкт є простим веб-додатком, створеним за допомогою фреймворку Django.

Проєкт реалізує роботу зі статтями: додавання, перегляд, редагування, видалення та стилізацію сторінок за допомогою статичних CSS-файлів.

Роботу виконано в межах лабораторної роботи №1 з дисципліни **«Високорівневі мови програмування та фреймворки»**.

## Виконані завдання

### Рівень 1, завдання 2

Реалізовано можливість додавання нових статей через адміністративну панель Django.

### Рівень 2, завдання 2

Створено веб-інтерфейс для редагування та видалення статей.

### Рівень 3, завдання 2

Додано стилі за допомогою статичних CSS-файлів для покращення зовнішнього вигляду веб-сайту.

## Функціональні можливості

У проєкті реалізовано:

- створення Django-проєкту;
- створення додатку `articles`;
- створення моделі `Article`;
- підключення моделі до адміністративної панелі Django;
- додавання статей через адмін-панель;
- відображення списку статей на веб-сторінці;
- редагування статей через веб-інтерфейс;
- видалення статей через веб-інтерфейс;
- підключення HTML-шаблонів;
- підключення CSS-стилів через статичні файли.

## Модель даних

У проєкті створено модель `Article`, яка використовується для збереження статей у базі даних.

Модель містить такі поля:

- `title` — заголовок статті;
- `text` — текст статті;
- `publication_date` — дата публікації.

## Адміністративна панель

Модель `Article` було зареєстровано в адміністративній панелі Django.

Через адміністративну панель адміністратор може:

- додавати нові статті;
- переглядати список створених статей;
- редагувати існуючі статті;
- видаляти статті;
- виконувати пошук і фільтрацію записів.

## Основні сторінки

| Адреса | Опис |
|---|---|
| `/` | список статей |
| `/edit/<id>/` | редагування статті |
| `/delete/<id>/` | видалення статті |
| `/admin/` | адміністративна панель Django |

## Запуск проєкту

### 1. Створення віртуального середовища

```bash
python -m venv .venv
```

### 2. Активація віртуального середовища

```bash
.venv\Scripts\activate
```

### 3. Встановлення Django

```bash
pip install django
```

### 4. Виконання міграцій

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Створення адміністратора

```bash
python manage.py createsuperuser
```

### 6. Запуск локального сервера

```bash
python manage.py runserver
```

## Посилання після запуску

Головна сторінка сайту:

```text
http://127.0.0.1:8000/
```

Адміністративна панель Django:

```text
http://127.0.0.1:8000/admin/
```
