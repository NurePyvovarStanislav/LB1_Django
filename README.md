# Django Articles Project

## Опис проєкту

Цей проєкт є простим веб-додатком, створеним за допомогою фреймворку Django.  
Основна мета проєкту — реалізувати роботу зі статтями: створення, перегляд, редагування, видалення та оформлення сторінок за допомогою статичних CSS-файлів.

Проєкт було виконано в межах лабораторної роботи №1 з дисципліни **«Високорівневі мови програмування та фреймворки»**.

## Функціональні можливості

У проєкті реалізовано:

- створення Django-проєкту та окремого додатку `articles`;
- модель `Article` для збереження статей;
- підключення моделі до адміністративної панелі Django;
- можливість додавання статей через адмін-панель;
- відображення списку статей на веб-сторінці;
- редагування статей через веб-інтерфейс;
- видалення статей через веб-інтерфейс;
- підключення HTML-шаблонів;
- підключення CSS-стилів через статичні файли.

Модель даних

У проєкті створено модель Article, яка містить такі поля:

title — заголовок статті;
text — текст статті;
publication_date — дата публікації.

Приклад моделі:

from django.db import models

class Article(models.Model):
    title = models.CharField(max_length=200, verbose_name="Заголовок")
    text = models.TextField(verbose_name="Текст статті")
    publication_date = models.DateTimeField(
        auto_now_add=True,
        verbose_name="Дата публікації"
    )

    def __str__(self):
        return self.title
Запуск проєкту
1. Клонування або відкриття проєкту

Відкрити папку проєкту у Visual Studio Code.

2. Створення віртуального середовища
python -m venv .venv
3. Активація віртуального середовища

Для Windows:

.venv\Scripts\activate
4. Встановлення Django
pip install django
5. Виконання міграцій
python manage.py makemigrations
python manage.py migrate
6. Створення адміністратора
python manage.py createsuperuser
7. Запуск локального сервера
python manage.py runserver

Після запуску сайт буде доступний за адресою:

http://127.0.0.1:8000/

Адміністративна панель доступна за адресою:

http://127.0.0.1:8000/admin/
Основні сторінки
/ — список статей;
/edit/<id>/ — редагування статті;
/delete/<id>/ — видалення статті;
/admin/ — адміністративна панель Django.
Висновок

У результаті виконання проєкту було створено простий Django-додаток для роботи зі статтями.
Було реалізовано модель даних, адміністративну панель, веб-інтерфейс для редагування та видалення статей, а також підключено статичні CSS-файли для покращення зовнішнього вигляду сайту.

Проєкт демонструє базові можливості Django та може бути використаний як основа для подальшого розвитку веб-додатку.
